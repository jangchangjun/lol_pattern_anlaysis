# 미드라이너의 매치 지표를 통한 게임의 승패 예측

## 1. 개요

리그 오브 레전드 게임은 5명의 팀원을 탑, 미드, 정글, 바텀의 네 라인으로 나누어 상대팀의 넥서스를 파괴하여 승리를 쟁취하는 게임이다.
4개의 라인 모두 게임에서 중요한 역할을 하지만 그 중에서도 미드라이너의 역할이 가장 중심을 맡고있어 게임의 결과에 가장 큰 영향을 미친다.
